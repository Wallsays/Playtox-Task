********************************************
********* Руководство по установке *********
********************************************
Перед запуском приложения необходимо изменить настройки доступа к базе.

А именно в файлах: 
	hibernate.cfg.xml и applicationContextPSQL.xml, 
которые находятся в src/main/resources изменить "DemoTask" на вашу БД в PostgreSQL, "manager" и "123" - логин, пароль соответственно.

Развертка приложений производилась в Jetty. 
Для запуска использовать любой из .bat файлов, в которых содержатся команды анологичные их названиям.

Далее перейти по адресу: http://localhost:8080/TestTask

Сам .WAR находится в каталоге target. Однако у меня запустить проект из скомпилированных файлов не удалось, хотя команды из .bat файлов при работе с исходниками выполняются успешно.

При импорте проекта в Eclipse можно также запустить проект через Start файл из каталога src/test/java/com/wicket (разворачивается также в Jetty).
Также можно будет запустить функцию заполнения тестовыми записями FillingContactBook из пакета com.tools, которая сгенерирует имена,фамилии и email'ы контактов; поле 'заметка' будет содержать порядковый номер записи при добавлении, т.е. они будут от 1 до 10000.

********************************************
***** Краткое руководство пользователя *****
********************************************

Приложение адресная книга, позволяет пользователям без аутенфикации просматривать, искать и менять адресные данные абонентов. Скриншот прилагается (./screenShot.png).

Доступны следующие поля контактов:
	- имя
	- фамилия
	- email
	- заметка
	- группа

Поля имя и фамилия обязательны для заполнения.

При поиске ищется четкое соответствие с одним из полей контакта.
